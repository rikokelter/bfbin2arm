# bfbin2arm: Bayesian Power for Two-Arm Binomial Bayes Factors

Functions for Bayesian power and sample size calculation in two-arm
binomial trials using Bayes factors for point-null and directional
hypotheses.

## Details

The package provides simulation-free and simulation-based methods to
calibrate Bayes factor designs in two-arm phase II trials with binary
endpoints, including power and type-I-error calculations and
visualization tools.
